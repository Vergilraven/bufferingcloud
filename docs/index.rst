bufferingcloud
**************
From begining:
The only purpose is for coding learning, not commercial, to copy and paste part of the code implementation.

| *bufferingcloud* provides a buffer while call slurm cluster.
| It wraps Plotly Dash in a custom object which hides infrastructure aspects from the model developer.

Additionally, it provides the following convenience functions:

- Automatic generation of input-form based on model's metadata (metadata.json)
- Client for working with bufferingcloud-service
- Client for accessing job-results
